# 프로젝트: 칵테일 추천 앱 "오늘 뭐 마시지" (가제)

## 개요
가진 재료 기반으로 만들 수 있는 칵테일을 추천하는 앱.
칵테일 입문 장벽을 낮추고, 대체재 안내로 단계별 입문 지원.

## 기술 스택
- Flutter (최신 stable)
- Supabase (인증, DB)
- 상태관리: Provider 또는 Riverpod
- 다국어: flutter_localizations + intl

## 데이터 소스
- Bar Assistant 오픈소스 데이터 활용 (github.com/bar-assistant/data)
- 500+ 칵테일 레시피, 250+ 재료 포함
- 대체재(substitute) 정보 포함
- MIT 라이선스

## 핵심 기능 (MVP)
1. 재료 등록 - 내가 가진 재료/도구 체크리스트
2. 칵테일 매칭 - 보유 재료로 만들 수 있는 칵테일 필터링
3. 레시피 상세 - 재료, 비율, 만드는 법
4. 대체재 표시 - "심플시럽으로 대체 가능" 등
5. "재료 1개만 더 있으면" - 추가 구매 유도

## 유저 플로우
1. 앱 실행 → 온보딩 (스킵 가능) → 회원가입/게스트
2. 홈 → 내 재료 등록 → 만들 수 있는 칵테일 리스트
3. 칵테일 선택 → 레시피 상세 (대체재 표시)

## 다국어 지원
- 영어 + 한국어
- 처음부터 i18n 구조로 세팅
- 칵테일/재료명은 대부분 원어 유지, UI 텍스트만 번역

## 디자인 방향
- Material 3 기본 컴포넌트 활용
- 미니멀, 깔끔하게
- 다크모드 지원

## 우선순위
P0: 재료 등록, 칵테일 매칭, 레시피 상세
P1: 대체재 표시, "재료 1개만 더 있으면"
P2: 온보딩, 소셜 로그인

## 시작 요청
1. Flutter 프로젝트 생성 및 기본 구조 세팅
2. Supabase 연결 준비
3. 다국어(i18n) 기본 세팅
4. Bar Assistant 데이터 가져와서 분석 후 데이터 구조 설계
5. 핵심 화면 구현: 홈, 재료 등록, 칵테일 리스트, 레시피 상세
